Escuela

Automaquillaje Automaquillaje

Clases personalizadas diseñadas para que aprendas técnicas sencillas para maquillarte y realzar tu belleza según su propio estilo, gustos, ocupación y actividades. Te mostraremos pequeños trucos para disimular o resaltar diferentes rasgos y prepararte para distintas ocasiones.

  • ·Duración: 3 clases de una hora cada una
  • ·
    Capacidad: Clases orientadas fundamentalmente a la práctica, con modalidad individual o grupal (¡podés venir con tu grupo de amigas!).
  • ·
    Materiales: El estudio proporciona los materiales pero aconsejamos que las alumnas traigan sus propios productos para enseñarles su uso adecuado.
Contactanos para tomar un curso de automaquillaje
Maquillaje Profesional Maquillaje Profesional

Curso dirigido a quienes quieren convertirse en verdaderos artistas del maquillaje brindando los conocimientos necesarios para desenvolverse con seguridad en trabajos con novias, sociales, artísticos y producciones para diferentes medios.

  • ·Duración: 40 clases de dos horas cada una
  • ·Capacidad: 4 a 6 alumnos
  • ·
    Materiales: el estudio proporciona los productos que se usan durante el dictado de sus cursos. Al comienzo del curso se hace entrega de un set de pinceles a cada alumno.
  • ·Clases: teóricas y prácticas.
  • ·Evaluación final con entrega de Certificado. Pasantías.
